Description: Respond to survey questions. Options for price should be 7.5, 18, 33, and 80, but the labels should be ($5-10), ($10-$25), ($26-$40), and $41+. Please reference the 'upick summary' doc for the actual values submitted vs the labels the user sees.

Method Type: POST

In: {"group": int,
"location_x": float,
"location_y": float,
"price": int,
"distance": int,
"time": int,
"preferences": [int, int, int]}

Out: {"success": bool, "data": "Responses submitted"}
